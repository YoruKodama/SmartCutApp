SmartCut App Android-приложение для управления умным слайсером на базе ESP32.
О проекте
SmartCut — это мобильное приложение, разработанное в рамках дипломного проекта. Приложение взаимодействует с умным слайсером, который выполняет нарезку продуктов на дольки, кубики и слайсы. Связь между приложением и устройством осуществляется через микроконтроллер ESP32.
